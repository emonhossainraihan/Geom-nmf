# Create a virtual env

```shell
> pip install virtualenv
> virtualenv env
> .\env\Script\activate
(env) > .\env\Script\python.exe -m pip install -r requirements.txt
```

OR

```shell
pip install numpy pandas scipy
```

How to active your env from cmd:

```
& <your_dir>/NMF/env/Scripts/Activate.ps1
```
